# 문제
상근이는 겨울방학을 맞아 N개국을 여행하면서 자아를 찾기로 마음먹었다. 

하지만 상근이는 새로운 비행기를 무서워하기 때문에, 최대한 적은 종류의 비행기를 타고 국가들을 이동하려고 한다.

이번 방학 동안의 비행 스케줄이 주어졌을 때, 상근이가 가장 적은 종류의 비행기를 타고 모든 도시들을 여행할 수 있도록 도와주자.

상근이가 한 국가에서 다른 국가로 이동할 때 다른 국가를 거쳐 가도(심지어 이미 방문한 국가라도) 된다. [baekjoon](https://www.acmicpc.net/problem/9372)

<br>

# 코드
```c++
#include "9372.h"
#include <iostream>
#include <vector>
#include <queue>

using namespace std;

void solve_9372() {
    int tc;
    cin >> tc;
    while (tc-- > 0) {
        int n, m;
        cin >> n >> m;

        vector<int> graph[n + 1];
        for (int i = 0; i < m; i++) {
            int a, b;
            cin >> a >> b;
            graph[a].push_back(b);
            graph[b].push_back(a);
        }

        int result = 0;
        vector<bool> isVisited(n + 1);
        queue<int> q;
        q.push(1);
        isVisited[1] = true;

        while (!q.empty()) {
            int node = q.front();
            q.pop();

            for (int connected : graph[node]) {
                if (!isVisited[connected]) {
                    q.push(connected);
                    isVisited[connected] = true;
                    result++;
                }
            }
        }
        cout << result << endl;
    }
}
```

<br>

# 풀이
BFS로 방문하면서, 새 간선을 지나가게 될 때마다 비행기 수(result)를 한 개씩 늘려준다.
