# 문제
[SWEA](https://www.swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AWd7sgDatsMDFAUh&categoryId=AWd7sgDatsMDFAUh&categoryType=CODE)

<br>

# 코드
```c++
#include "6719.h"
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

void solve_6719() {
    int tc;
    cin >> tc;

    for (int i = 1; i <= tc; i++) {
        int n, k;
        cin >> n >> k;

        vector<double> v;
        for (int j = 0; j < n; j++) {
            double tmp;
            cin >> tmp;
            v.push_back(tmp);
        }

        // 내림차순
        sort(v.begin(), v.end(), greater<double>());

        double result = 0;
        for (int l = k - 1; l >= 0; l--) {
            result = (result + v[l]) / 2.0;
        }

        printf("#%d %.6f", i, result);
        if (i != tc) cout << endl;
    }
}
```

<br>

# 풀이
강좌를 수준에 대해 내림차순으로 정렬한 후, 가장 상위의 k개에 대해서 수준이 낮은 것 부터 시청한다.
