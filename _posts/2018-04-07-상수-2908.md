# 문제
상근이의 동생 상수는 수학을 정말 못한다. 상수는 숫자를 읽는데 문제가 있다. 이렇게 수학을 못하는 상수를 위해서 상근이는 수의 크기를 비교하는 문제를 내주었다. 상근이는 세 자리 숫자 두 개를 칠판에 써주었다. 그 다음에 크기가 큰 수를 말해보라고 했다. 상수는 수를 다른사람과 다르게 거꾸로 읽는다. 예를 들어, 734과 893을 칠판에 적었다면, 상수는 이 수를 437과 398로 읽는다. 따라서, 상수는 두 수중 큰 수인 437을 큰 수라고 말할 것이다. 두 수가 주어졌을 때, 상수의 대답을 출력하는 프로그램을 작성하시오. [baekjoon](https://www.acmicpc.net/problem/2908)

<br>

# 코드
```java
import java.util.Scanner;

public class 상수_2908 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A = sc.nextInt(), B = sc.nextInt();
        sc.close();

        int newA = 0, newB = 0, tmp = 100;
        for (int i = A; i > 0; i /= 10) {
            newA += i % 10 * tmp;
            tmp /= 10;
        }
        tmp = 100;
        for (int i = B; i > 0; i /= 10) {
            newB += i % 10 * tmp;
            tmp /= 10;
        }
        System.out.println(newA > newB ? newA : newB);
    }
}
```

<br>

# 풀이
A와 B의 1의자리 수부터 역행으로 가면서 자리수를 반대로 바꿔준다. 문제에서 입력조건이 세자리 숫자여서 간편하게  tmp값을 100으로 줄 수 있었다. 새로 바뀐 A와 B중에 큰 값을 출력한다.
